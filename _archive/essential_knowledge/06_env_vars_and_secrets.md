# Environment Variables And Secrets

## Known Variables

- `GEMINI_API_KEY`
- `ECONOMIKA_ADMIN_API_KEY`
- `ECONOMIKA_ACCOUNT_ID`
- `CENTRAL_AI_BASE_URL`
- `CENTRAL_PUBLISHING_HUB_BASE_URL`
- `GRAPH_API_VERSION`
- `IG_MEDIA_PROCESSING_TIMEOUT_SECONDS`
- `IG_MEDIA_PROCESSING_POLL_SECONDS`
- `ECONOMIKA_X_SOURCE`
- `ECONOMIKA_DISCOVERY_MODE`
- `ECONOMIKA_ENABLE_X_SCOUT`

## Rules

- do not commit `.env`;
- do not commit X cookies;
- do not commit `client_secrets.json`;
- do not commit OAuth pickle files;
- do not commit browser profiles;
- do not commit debug dumps containing API responses or cookies.
